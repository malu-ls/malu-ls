## 💫 Sobre mim

Sou desenvolvedora de software com foco em **front-end**, apaixonada por transformar ideias em experiências visuais e funcionais. Minha trajetória na tecnologia começou cedo — e desde então venho construindo soluções que fazem diferença de verdade.

- 🎯 **Foco atual:** Angular, React e arquitetura front-end escalável
- 🌱 **Sempre aprendendo:** me desenvolvo continuamente para ter uma visão completa dos projetos
- ⚡ **Minha abordagem:** praticidade, organ para otimizar cada processo
- 🎵 **Fora do código:** música, jogos, livros e séries — sim, sou geek de carteirinha

> *"Acredito no poder da tecnologia para trasoluções com impacto real."*
