## Sobre mim

Desenvolvedora full stack com raízes no front-end e olhar cada vez mais voltado para **integração com IA**. Gosto de construir produtos completos, da interface ao dado e de encontrar onde a inteligência artificial pode tornar uma solução mais inteligente de verdade.

- 🧠 Trabalho com APIs de IA (OpenAI, Claude, Gemini) e banco de dados vetorial com **pgvector + Supabase**
- ⚡ Stack principal: **TypeScript · React · Angular · Node.js · Laravel**
- 🎯 Foco em código limpo, produto funcional e experiências que fazem sentido
- 🎮 Fora do código: música, jogos e séries — sim, sou geek
